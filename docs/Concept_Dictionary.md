# 概念字典（术语对齐）

## Task（任务）
一套从“参考图 + 需求参数”到“Shot List + 多张输出图 + 修复版本”的完整作业单元。

## Refs（参考图）
- 图A~图N：衣服参考图（按上传顺序自动编号）
- face_ref：模特头像参考图（可选，用于保持同一人）

## Garment Profile（服装识别档案）
只根据参考图可见事实总结的结构化字段集合；看不见必须写“无法从图片确认”。

## GARMENT LOCK（衣服锁定点）
从参考图可确认、且最容易被生图跑偏的“必须保持一致”的细节清单；可人工编辑。

## Shot（镜头/单张计划）
一张最终图的拍摄/动作/镜头/构图/光线描述 + 对应英文 Prompt（可复制）。

## Render（生成/渲染）
按某个 Shot 的 Prompt 对参考图进行生成，产出一张图；每次生成都应有记录与可重试能力。

## Review（质检）
人工对比“参考图 vs 输出图”，判断是否跑偏衣服细节。

## Fix（修复/改图）
在不改变人脸/姿势/场景/光线/镜头的前提下，仅修改衣服细节，使其更贴近参考图与 GARMENT LOCK。
